## 擬似言語IDE（科目B向け・簡易実行器）

GitHub Pages で公開して使う、科目Bっぽい擬似言語の簡易IDEです。  
疑似コードを入力して「実行」または「ステップ」で動かし、右側に変数と出力を表示します。

---

## 公開URL（GitHub Pages）

GitHub Pages を有効化すると、通常は次の形式のURLでアクセスできます。

- `https://<ユーザー名>.github.io/<リポジトリ名>/`

例: `https://example.github.io/Pseudo-ide/`

---

## GitHub Pages へのデプロイ手順

### 1) リポジトリに配置する
このリポジトリのルートに、少なくとも次のファイルがある状態にします。

- `index.html`（アプリ本体）
- `sort_examples.html`（サンプル閲覧用）
- `sort_examples.txt`（サンプル本文）

### 2) Pages を有効化する
GitHub のリポジトリ画面で:

- **Settings** → **Pages**
- **Build and deployment**
  - Source: **Deploy from a branch**
  - Branch: **main** / **root**（または公開したいブランチ/ディレクトリ）

保存後、しばらくすると公開URLが表示されます。

## 使い方

- **実行**: 最初から最後まで実行
- **ステップ**: 1命令ずつ実行
- **リセット**: 再パースして実行状態を初期化
- **使い方（先頭へ戻る）**: 右側「使い方サンプル」を初期表示に戻す
- **sort_examples を開く**: サンプル集を別タブで表示（GitHub Pages 上で文字化けしにくい）

## 対応している文法（要点）

### 代入
- `x ← 10`
- `x = 10` は **代入としては未対応**（比較として扱われます）  
  → 代入は `←` を使ってください

### 条件分岐（IPA形式：括弧必須）
- `if (条件式)` / `elseif (条件式)` / `else` / `endif`
- 従来形式 `if 条件式 THEN` も利用可

### ループ（IPA形式：括弧必須）
- `for (i ← 1 to N)` / `endfor`
- `for (i ← N downto 1)` / `endfor`
- `while (条件式)` / `endwhile`
- 従来形式 `for i ← 1 TO N` も利用可

### 入出力
- `print expr`
- `input x`（右下の入力欄から改行区切りで供給）

### 配列（1始まり）
- 参照: `A[i]`
- 代入: `A[i] ← expr`
- 初期化: `A ← [ 5, 2, 4, 1, 3 ]`
- 長さ: `N ← length(A)`  
  ※「最大添字」を要素数として扱います（1始まり前提）

### 手続き（再帰対応）
- 定義: `procedure Name(a, b)` ... `endprocedure`
- 呼び出し: `call Name(x, y)`
- 戻り: `return`（値なし）

---

## 比較演算子について（重要）

- **OK**: `=`, `!=`, `<`, `<=`, `>`, `>=`
- **注意**: `≠` は式として認識されずエラーになる可能性があります  
  → `≠` の代わりに `!=` を使ってください  
  → さらに確実にするなら `NOT (a = b)` の形にしてください

## 仕様メモ

- 未代入の変数は `0` 扱い（科目B寄せ）
- 配列添字は **1以上の整数**
- 変数と配列の同名は禁止（例: `A` をスカラとしても配列としても使うのは不可）
- 実行中にコードを編集すると「未リセット」表示になり、実行/ステップ時に自動でリセットされます
